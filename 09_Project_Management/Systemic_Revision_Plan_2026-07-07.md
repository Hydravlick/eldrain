# Системная редактура Элдрейна — план внедрения

> **Для агентных исполнителей:** выполнять задачи по независимым файловым контурам; не использовать Git; после каждого контура проверять термины, ссылки и утверждённые границы канона.

**Цель:** встроить в живой GDD согласованную терминологию архетипов, адресный бартер через карту, Круг Заслона, пищевую тревогу и тёмную историю первых Жаб, одновременно сделав ключевые тексты легче для чтения без потери правил.

**Архитектура:** единый канон остаётся распределён по ответственностям: правила экономики — в `06_Economy_Loot`, отображение и жизненный цикл адресов — в `08_World_Generation`, общественные процедуры — в `03_Factions_Societies`, историческая причина — в `02_World_Lore`, игроковые архетипы — в `04_Player_Entities`. Реестры хранят стабильные ID; повествовательные страницы объясняют смысл и опыт, не дублируя схемы.

**Формат:** Markdown + YAML frontmatter + Obsidian links + Dataview-совместимые inline fields.

**Итог 2026-07-08:** внедрение по контурам завершено. Ниже чеклист сохранён как аудитная карта выполненных правок.

## Общие ограничения

- Последние утверждения автора имеют приоритет над старым текстом и `CLAUDE.md`.
- `CLAUDE.md` и `Foodchain.md` — источники намерения, не самостоятельный канон.
- Бартер и мирные действия происходят вне аномалии через POI-диорамы/пины карты.
- Постоянные пины центра дают надёжный минимум; временные пины внешних лепестков дают специализацию, а не универсально лучшую ценность.
- Формула адресного бартера детерминирована: извлечённый состав + доступный адрес + окно цикла -> показанный результат.
- Аномалия остаётся пространством добычи, переноса и эвакуации; обычный крафт в рейде не вводится.
- Круг Заслона защищает паузу и участников, но не определяет вину и не становится полицией.
- Еда, вода, экстренная медицина и минимальный кров не могут быть отняты как наказание.
- Труп не является топливом сам по себе: ранние Хранители использовали краткий посмертный информационный след без согласия.
- Тема разумной плоти остаётся редким преступлением и тревогой происхождения, а не нормальной гастрономической экономикой.
- Технические ID `assault`, `support`, `scout` пока сохраняются ради схем; игроковые названия — Авангард, Технократ, Странник.
