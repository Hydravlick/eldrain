# Система Навыков и Архетипов

## 1. Структура Навыков (2+1)
Каждый персонаж обладает фиксированным набором из 3 слотов.

* **2 Активных Навыка:**
    * Требуют энергии (**Магические Батареи**) или специфических расходников.
    * Имеют Cooldown.
    * Привязаны к клавишам `Q` и `E`.
* **1 Пассивный Навык:**
    * Постоянный эффект или аура.
    * Не требует ресурсов.
    * Определяет стиль игры (например, "Регенерация при беге" или "Бонус к урону из тени").

## 2. Формирование Навыков
Навыки не выбираются из списка, а **складываются** из тегов персонажа при генерации.
* `Раса` (База) + `Специализация` (Основной скилл) + `Тег` (Модификатор).

---

## 3. Архетипы Навыков
Все способности в игре делятся на функциональные категории для ясности баланса.

### Атака (Offense)
* **DirectDamage (DD):** Прямой урон по одной цели. Требует точного наведения (Skillshot). *Пример: Эфирная Стрела.*
* **AreaDamage (AoE):** Урон по площади. Дорогой по мане. *Пример: Граната Бездны.*
* **Trap (Ловушки):** Контроль территории. Отложенная активация. *Пример: Рунная Мина.*

### Контроль и Поддержка (Support)
* **Control (CC):** Стан, замедление, ослепление. Критично для Танков.
* **Buff/Debuff:** Усиление союзников или ослабление врагов (снижение брони).
* **Heal:** Восстановление HP (редкий и дорогой архетип).

### Утилиты (Utility)
* **Mobility:** Рывки, телепорты, крюк-кошка.
* **Defense:** Персональные щиты, стены.

---

## 4. Ролевая Система (Class Roles)
Комбинация архетипов формирует роль в отряде:

| Роль | Архетипы | Задача |
| :--- | :--- | :--- |
| **Assault (Штурм)** | DD + Mobility | Агрессия, флангование. |
| **Tank (Танк)** | Defense + Control | Впитывание урона, инициация боя. |
| **Support (Саппорт)** | Heal + Buff | Удержание команды на плаву. |
| **Scout (Разведка)** | Trap + Mobility | Сбор информации, контроль проходов. |
| **Specialist (Спец)** | Mix | Уникальные механики (стелс, взлом, призыв). |

# Система: Боевая Механика (Combat Physics)

## 1. Приоритет Анимации (Commitment System)
* **Замах:** Инициация атаки блокирует возможность передвижения или смены действия до завершения активной фазы (Recovery Frames).
* **Инерция:** Скорость поворота персонажа во время удара ограничена.

## 2. Коллизия Оружия (Wall Collision)
* Оружие имеет физический коллайдер.
* При контакте со статическим объектом (стена, колонна) во время фазы атаки:
    1. Анимация прерывается.
    2. Накладывается микро-стан (Stagger) на атакующего.
    3. Проигрывается анимация отскока.

## 3. Огнестрельное Оружие (Sidearms)
* Классификация: Вспомогательный предмет экипировки (не навык).
* **Механика:**
    * Смена оружия занимает длительное время (Equip Time > 2.0 sec).
    * Перезарядка: Мануальная, по одному патрону (Loop Animation).
    * Тип урона: Raycast (точечный), высокий пробивающий потенциал для поражения Уязвимых Зон (Gaps).
