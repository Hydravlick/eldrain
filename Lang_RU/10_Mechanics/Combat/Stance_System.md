# Система Стоек (Stance System)

## 1. Две стойки

### Стойка "Беглец" (Runner Stance)
**Активация:** `X` или автоматически при спринте
**Анимация перехода:** 0.5 сек (персонаж убирает оружие в ножны/на спину)

| Параметр      | Значение              | Визуал                             |
| :------------ | :-------------------- | :--------------------------------- |
| Скорость      | 100%                  | Руки свободны, оружие за спиной    |
| Стамина drain | 0 (восстанавливается) | -                                  |
| Атака мили    | Невозможна            | -                                  |
| Стрельба      | Невозможна            | -                                  |
| Блок          | Невозможен            | -                                  |
| Защита головы | Открыта               | Голова НЕ прикрыта                 |
| Защита спины  | **Пластины на спине** | Если есть наплечники, они на спине |
| Лут           | Быстрый (1 сек)       | Можно брать на бегу                |

**Тактика использования:**
- Бегство из боя
- Быстрый грабёж контейнеров (Munchkin mode)
- Паркур (прыжки требуют свободных рук)

**Риск:**
Если тебя настигли в этой стойке, ты **не успеешь** перейти в боевую (анимация 0.5с → тебя убьют за это время)

---

### Стойка "Дуэлянт" (Combat Stance)
**Активация:** `X` или автоматически при попытке атаки/блока
**Анимация перехода:** 0.5 сек (персонаж достаёт оружие)

| Параметр | Значение | Визуал |
|:---------|:---------|:-------|
| Скорость | 60% | Оружие в руках, прицел готов |
| Стамина drain | Восстановление -50% | Тяжёлое дыхание |
| Атака мили | 100% | Доступны все комбо |
| Стрельба | 100% (ADS доступен) | Можно прицеливаться |
| Блок | Доступен (RMB) | Щит/оружие поднято |
| Защита головы | **Приподнятое оружие прикрывает лицо** | Пассивная защита |
| Защита груди | **Пластины развёрнуты вперёд** | Наплечники защищают сердце |
| Лут | Медленный (3 сек) | Нужно убрать оружие |

**Тактика использования:**
- Бой
- Прицельная стрельба
- Танкинг (подставить бронированное плечо под выстрел)

**Риск:**
Медленное передвижение = лёгкая мишень для снайперов

## 2. FPS Механика: "Танкинг без TPS"

**Проблема:** В FPS игрок не видит своё тело. Как "подставить плечо"?

**Решение: Auto-Tanking при блоке**

Когда игрок зажимает `RMB` (блок):
1. Камера слегка сдвигается влево/вправо (5-10°)
2. Модель персонажа **автоматически разворачивает** бронированное плечо к центру экрана
3. Hitbox пластины **расширяется** на 20% перед игроком

**Визуальный фидбек:**
- На экране появляется силуэт пластины (UI overlay) с индикатором прочности
- При попадании в пластину: искры, звук металла, экран вздрагивает (camera shake)

**Пример:**
```
Игрок видит врага впереди → Зажимает RMB (блок) → 
→ Камера чуть сдвинулась вправо → На экране появился силуэт наплечника →
→ Враг стреляет → Попадание в пластину → 0 урона, искры, Impact (игрока отбросило на 1м)
```
