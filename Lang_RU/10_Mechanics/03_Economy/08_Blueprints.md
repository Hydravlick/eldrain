# Система Чертежей (Blueprints)

## 1. Типы Чертежей
Чертеж — это физический предмет, необходимый для разблокировки большинства сложных рецептов.

| Тип                      | Иконка          | Поведение при крафте                                               |
| :----------------------- | :-------------- | :----------------------------------------------------------------- |
| **Consumable (Схема)**   | Бумажный свиток | **Удаляется** (-1 шт). Одноразовый крафт.                          |
| **Durable (Инструкция)** | Планшет/Папка   | **Уменьшается прочность** (Has Durability). Хватает на 2-6 циклов. |
| **Permanent (Матрица)**  | Кристалл данных | **Возвращается** игроку. Вечное знание.                            |

## 2. Логика Использования
Система ищет необходимый чертеж в следующем приоритете:
1.  **Слот крафта:** Если игрок положил чертеж вручную в интерфейс.
