# Износ и Деградация (Item Degradation)

## 1. Физика Износа
В мире Элдрейна материя стремится к распаду. Прочность (`Durability`) — это не просто здоровье предмета, это его сопротивление Энтропии.

### Причины Износа:
1.  **Использование:** Каждый выстрел снижает прочность оружия на `0.1`. Каждый удар по броне снижает прочность брони.
2.  **Смерть:** При смерти (даже с возвратом по страховке) экипировка теряет **15-20%** прочности сразу.
3.  **Энтропийный Шторм:** Нахождение в зоне после таймера эвакуации быстро разрушает всё снаряжение.

## 2. Последствия (Breakpoints)
| Прочность % | Эффект |
| :--- | :--- |
| **100-50%** | Штатная работа. |
| **49-20% (Worn)** | **Снижение характеристик.** Броня дает меньше защиты, оружие теряет точность. |
| **< 20% (Critical)** | **Заклинивание (Jamming).** Оружие может заклинить при перезарядке. |
| **0% (Broken)** | Предмет **неактивен**. Его нельзя использовать, но можно починить (дорого) или разобрать на Скрап. |

## 3. "Усталость Металла" (Max Durability Loss)
Обычный ремонт у NPC восстанавливает текущую прочность, но снижает максимальную.
* *Формула:* `New_Max_Durability = Old_Max_Durability - (Repair_Amount * 0.1)`
* *Пример:* Вы чините винтовку с 0 до 100. Её новый предел станет 90/90.
* *Решение:* Чтобы починить предмет без потери качества, нужно использовать редкие **"Наборы Стабилизации"** (Stabilization Kit) или платить премиум-валютой (Скриптами Фракций).
