# Механика: Резонанс (Эфирный Шум)

## 1. Концепция
Резонанс — это вибрация чужеродной материи внутри Аномалии. Монстры чувствуют не ваши шаги, а вашу "ценность" и "чужеродность".
* **Кто слышит:** Высшие Сущности, Охотники, Аномальные вихри.
* **Свойство:** Проходит сквозь стены. От него нельзя спрятаться за укрытием, только сбросить лут.

## 2. Расчет Личного Резонанса
$$PersonalRes = (InventoryPrice \times SyncMult) + BodyModifiers$$

### Модификаторы Тела (Body Tags)
Врожденные свойства персонажа влияют на его заметность.

* **Tech Tags (Импланты):** Увеличивают Резонанс. Технология конфликтует с магией.
    * `[tag:: piston_arm]` -> **+300 Rez** (Громкий).
    * `[tag:: alchemical_eye]` -> **+150 Rez**.
* **Mutation Tags (Мутации):** Снижают Резонанс. Вы становитесь частью экосистемы.
    * `[tag:: greyscale]` -> **-500 Rez** (Маскировка).
    * `[tag:: hollow_bones]` -> **-200 Rez**.

## 3. Командный Резонанс (Group Formula)
Используется правило **"Громоотвода"**.

$$GroupThreat = Max(Member_1...N) + (\sum(Others) \times 0.2)$$

* **Тир Мобов:** Определяется по `Max` (самому "дорогому" игроку). Нельзя занизить сложность, взяв в группу новичка.
* **Плотность Спавна:** Определяется общей суммой. Чем больше людей, тем больше мобов придет на "зов", но они будут того уровня, который соответствует Лидеру по угрозе.
