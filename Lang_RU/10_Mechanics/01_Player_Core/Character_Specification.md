# Спецификация: Био-Адаптивное Ядро

## 1. Философия Сборки
Персонаж состоит из **Биологического Носителя** (Раса) и **Класса** (Специализация).

### Формула Генерации
Итоговый геймплей определяется взаимодействием биологии и технологий:

$$Entity = (Race\_Physiology \times Artifact\_Core) + Tags\_Modifiers$$

1.  **Race (Раса):** Определяет **Пассивный Навык** (метод выживания), Базовые статы и Физику передвижения.
2.  **Artifact (Артефакт/Класс):** Определяет **Активные Навыки** (Q/E) и Роль в отряде (Танк, ДД, Саппорт).
3.  **Weapon (Оружие):** Генерируется автоматически на стыке Расы и Артефакта (см. раздел "Адаптивный Арсенал").
4.  **Tags (Теги):** Модифицируют навыки или открывают доступ к запрещенному оружию.

## 2. Адаптивный Арсенал (Adaptive Loadout)
Оружие не выбирается из общего списка. У персонажа есть список допустимого надеваемого оружия из общего списка.

| Артефакт (Роль) | Раса (Носитель) | Полученное Оружие | Обоснование |
| :--- | :--- | :--- | :--- |
| **Страж (Танк)** | **Лиса (Agile)** | *Серпы Парирования* | Не может держать удар. Парирует и контратакует. |
| **Страж (Танк)** | **Медведь (Heavy)** | *Ростовой Молот* | Использует массу для контроля зоны и "Гипер-брони". |
| **Стрелок (ДД)** | **Лиса (Agile)** | *Парные ПП* | Высокая мобильность, стрельба на бегу (Run & Gun). |
| **Стрелок (ДД)** | **Медведь (Heavy)** | *Ручная Пушка* | Станковая стрельба, отдача требует массы тела. |

## 3. Расчет Характеристик (Stats Calculation)
Итоговые цифры складываются из базы Расы и бонусов Артефакта/Тегов.

| Параметр | Формула |
| :--- | :--- |
| **Max HP** | `Race_Base_HP + Artifact_Modifier + (CON * 10)` |
| **Speed** | `Race_Base_Speed * (100% - Armor_Penalty)` |
| **Stamina** | `Race_Base_Stam + (END * 2)` |
| **Stability** | `Race_Weight + Armor_Weight` (Сопротивление опрокидыванию) |

## 4. Слоты (Hardpoints)
* **Body Core:** Слот для Артефакта (Специализации).
* **Genetic Slot:** Слот Расы (неизменяемый).
* **Memory Slots (3-5):** Слоты под **Теги** (Перки/Мутации).
