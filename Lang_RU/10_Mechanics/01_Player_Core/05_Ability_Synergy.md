# Система Синергий (Ability Fusion)

## 1. Принцип Слияния
Навыки персонажа не берутся из списка "Навыки Вора". Они генерируются на пересечении Расы и Специализации.

### Активные Навыки (Q / E)
Каждый Специалист имеет **Шаблон Навыка (Blueprint)**. Раса наполняет этот шаблон содержанием.

> **Пример: Шаблон Специалиста "Рывок" (Dash)**
> * *Раса Крыса:* **Tactical Slide.** Подкат с возможностью стрельбы.
> * *Раса Ёж:* **Battering Ram.** Таран телом. Сбивает врагов с ног, наносит урон, но нельзя стрелять.
> * *Раса Белка:* **Phase Shift.** Телепорт сквозь врагов/стены.

## 2. Пассивный Навык (Trait)
Определяется **Расой**, но модифицируется **Спеком**.

> **Пример: Расовая пассивка "Регенерация" (Жаба)**
> * *Специалист Воин:* Регенерация работает всегда.
> * *Специалист Маг:* Регенерация потребляет Ману (Обмен ХП на Ману).
> * *Специалист Техник:* Регенерация заменена на "Ремонт Брони" (кожа стала металлом).

## 3. Матрица Синергий
Уникальная способность (Hidden Ability) может открываться только при совпадении условий тегов.

$$Ability = (Race\_Tag * Class\_Tag) + List[tags]$$

Примеры уникальных комбинаций:
* **Squirrel (Белка) + Guard (Страж) = Тень-Защитник:** Рывок к союзнику из инвиза для блока удара.
* **Rattus (Крыса) + Technocrat (Технократ) = Чумной Доктор:** Превращает дебаффы союзников в баффы.

## 4. Структура Навыков (2+1)
* **2 Активных Навыка (Q/E):** Требуют энергии (Магические Батареи).
* **1 Пассивный Навык:** Постоянный эффект или аура.
