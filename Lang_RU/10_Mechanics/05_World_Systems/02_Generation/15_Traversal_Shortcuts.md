# Слой Связности (Connectivity Layer)

## 1. Цель Системы
После генерации зданий алгоритм "Connector" анализирует соседство объектов, чтобы создать нетривиальные пути отхода и вертикальный геймплей.

## 2. Типы Связей

### Rooftop Bridges (Крыши)
Если два высоких здания стоят рядом (дистанция < X метров), между ними с шансом 40% генерируется переправа:
* Доска (разрушаемая).
* Труба (узкая, требуется баланс).
* Капитальный мостик.

### Breaches (Проломы)
Если глухая стена здания граничит с переулком, в ней может быть сгенерирован "Пролом":
* Взорванная кладка, изначально закрытая досками.
* Игрок может разрушить доски выстрелом/ударом, чтобы создать короткий путь (Shortcut) внутрь.

### Zip-Lines (Тросы)
От высокой точки (High Ground) к низкой (Low Ground) протягивается трос для быстрой эвакуации.
* *Условие:* Чистая линия видимости (Raycast) без препятствий.
