# Цикл Жизни Сервера и Гейт-Проверка (World Lifecycle)

## 1. Структура Сессии (6-Hour Cycle)
Один серверный инстанс живет ровно **6 часов**. Внутри этого времени мир проходит 3 фазы сложности (Tier Shift). Игрок может зайти и выйти в любой момент, пока инстанс жив.

| Тайминг (Время сервера) | Фаза Мира | Состояние Лута | Сложность Мобво |
| :--- | :--- | :--- | :--- |
| **00:00 - 02:00** | **Tier 1 (Stabilization)** | Базовый | Scavengers / Rats |
| **02:00** | *Gate Check I (Сдвиг)* | *Респавн лута* | *Мутация* |
| **02:00 - 04:00** | **Tier 2 (Escalation)** | +Quality / +Quantity | Mercenaries / Mutants |
| **04:00** | *Gate Check II (Сдвиг)* | *Респавн лута* | *Эволюция* |
| **04:00 - 06:00** | **Tier 3 (Entropy Storm)** | **High Value / Artifacts** | Bosses / Horrors |
| **06:00** | **THE WIPE (Коллапс)** | Полное уничтожение | Смерть всего живого |

---

## 2. Механика Перехода: Гейт-Проверка (Gate Check)
Смена фазы (например, с Т1 на Т2) — это не просто смена цифр. Это мощный **Выброс Энтропии**.
В момент "02:00" и "04:00" по всему серверу проходит ударная волна.

### Формула Выживания
Чтобы остаться на сервере при повышении сложности, игрок должен "танковать" Выброс своим снаряжением.

> **Condition:** `(Current_HP * Gear_Resonance_Resist) >= Entropy_Pulse_Damage`

1.  **Смерть (Death):** Если защита слабая — игрок погибает мгновенно. Персонаж считается "растворившимся", лут теряется.
2.  **Обморок (Blackout):** Если защита выдержала удар — игрок получает статус **`Unconscious`** (Бессознательное состояние).
    * **Эффект:** Экран гаснет. Игрок теряет управление на 2-5 минут (пока мир "перестраивается").
    * **Пробуждение:** Игрок приходит в себя отброшенным к ближайшему укрытию. Вокруг уже **Tier 2** мир: новые монстры, обновленные ящики.

## 3. Динамика Респавна
Помимо глобальных сдвигов, работает локальное наполнение.

* **При Сдвиге (2:00, 4:00):** Все контейнеры на карте принудительно обновляются (Reroll). Закрытые двери запираются снова.
* **Между Сдвигами:** Когда контейнер считается "Опустошенным". Он снова наполнится лутом ТОЛЬКО если выполнены **оба** условия:
	1.  **Timer:** Прошло > 10-15 минут с момента последнего обыска.
	2.  **Empty Zone:** В радиусе **X метров** (напр. 60м) нет ни одного игрока.

## 4. Эвакуация vs Жадность
Это создает главный выбор сессии:
* *Таймер 01:50.* У тебя полный рюкзак Т1 лута.
* **Выбор А:** Уйти сейчас (Гарантированный доход).
* **Выбор Б:** Спрятаться, пережить удар (рискуя потерять всё, если броня слабая) и очнуться в Т2, чтобы набить рюкзак более дорогим лутом.
