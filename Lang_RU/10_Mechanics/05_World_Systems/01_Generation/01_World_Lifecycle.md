# Жизненный Цикл Сервера (Server Lifecycle)

## 1. Структура Сессии (6-Hour Cycle)
Один серверный инстанс живет ровно **6 часов**. Внутри этого времени мир проходит 3 фазы сложности (Tier Shift). Игрок может зайти и выйти в любой момент, пока инстанс жив.


| Тайминг (Время сервера) | Фаза Мира | Состояние Лута | Сложность Мобво |
| :--- | :--- | :--- | :--- |
| **00:00 - 02:00** | **Tier 1 (Stabilization)** | Базовый | Scavengers / Rats |
| **02:00 - 04:00** | **Tier 2 (Escalation)** | +Quality / +Quantity | Mercenaries / Mutants |
| **04:00 - 06:00** | **Tier 3 (Entropy Storm)** | **High Value / Artifacts** | Bosses / Horrors |
| **06:00** | **THE WIPE (Коллапс)** | Полное уничтожение | Смерть всего живого |

## 2. Эвакуация vs Жадность
Это создает главный выбор сессии. Например, на таймере 01:50 у игрока полный рюкзак Т1 лута:
* **Выбор А:** Уйти сейчас (Гарантированный доход).
* **Выбор Б:** Пережить "Гейт-Проверку" (с риском смерти) и очнуться в Т2, чтобы собрать более дорогой лут.
