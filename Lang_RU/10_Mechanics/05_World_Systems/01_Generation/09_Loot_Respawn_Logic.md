# Динамика Респавна Лута

## 1. Глобальный Реролл
В моменты сдвига фаз (02:00, 04:00) все контейнеры на карте принудительно обновляются, а закрытые двери запираются снова.

## 2. Локальное Наполнение
Между сдвигами контейнер наполняется заново только при выполнении **двух условий**:
1.  **Timer:** Прошло > 10-15 минут с момента последнего обыска.
2.  **Empty Zone:** В радиусе X метров (напр. 60м) нет ни одного игрока.

## 3. Lazy Load (Ленивая загрузка)
Лут генерируется сервером только в момент открытия ящика ("Открываю ID 555" -> Сервер кидает кубик -> Спавнит нож). Это экономит память.
