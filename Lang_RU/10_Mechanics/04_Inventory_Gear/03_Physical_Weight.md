# Механика: Физический Вес (Physical Weight)

## 1. Разделение понятий
Вес предмета в килограммах (`Item.Weight`) влияет **только** на физику тела персонажа. Он **не** влияет на заметность для монстров (см. `Resonance_Value`).

## 2. Параметры Влияния
Суммарный вес инвентаря изменяет следующие характеристики:
1.  **Расход Стамины (Stamina Drain):** Линейное увеличение затрат на бег и действия.
2.  **Инерция (Inertia):** Время разгона и торможения. Тяжелый персонаж "плывет" при остановке.
3.  **Высота Прыжка:** При перегрузе прыжок блокируется.
4.  **Акустический Шум:** Громкость звука шагов (слышимость для ушей, но не для "чутья" аномалий).

## 3. Стадии Нагрузки (Load Stages)
Рассчитывается от параметра `MaxCarryWeight` ([TRQ]).

| Стадия | % Веса | Эффекты |
| :--- | :---: | :--- |
| **Light** | 0-40% | Нет штрафов. Макс. реген стамины. |
| **Combat** | 41-70% | Незначительное снижение регена стамины (-10%). |
| **Heavy** | 71-100% | Инерция ++. Спринт расходует стамину x1.5. Громкие шаги. |
| **Overload** | >100% | Спринт отключен. Прыжки отключены. Только шаг. |
