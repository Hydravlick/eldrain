# Торговля: Система Чертежей (Blueprints)

## 1. Типы Чертежей
Чертеж — это физический предмет, необходимый для разблокировки большинства рецептов.

| Тип | Описание | Поведение после крафта |
| :--- | :--- | :--- |
| **Consumable (Single)** | Одноразовая схема. | **Удаляется** (-1 шт). |
| **Durable (Stack)** | Пачка инструкций. | Уменьшается прочность/стак. |
| **Permanent (Perm)** | Твердая копия/Книга. | **Возвращается** игроку. |

## 2. Проверка Наличия
Система ищет чертеж в:
1.  Слоте крафта (если игрок положил его туда вручную).
2.  Инвентаре.(?)
3.  Схроне.(?)
4.  *Библиотеке Хаба* (если реализована механика "Изученных рецептов").(?)
