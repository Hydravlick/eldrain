# Элдрейн: Дизайн-Документ (Master GDD)

## 1. Общая Информация

* **Жанр:** Hardcore FPS Extraction RPG / Dungeon Crawler

* **Камера:** 1st Person (с Body Awareness)

* **Режим:** 1-3 игрока (Кооп) vs PvPvE

* **Боевая система:** Stance-Based Combat (Стойки) + Композитная Броня
 ---
 
## 2. High Concept: "Ключ и Замок"

### Ключевая особенность (USP):

**Система "Броня как Геометрия" (Armor is Geometry)**

- Защита определяется **физическим покрытием** модели пластинами, а не числовыми показателями

- В FPS игрок управляет уязвимостью через **систему стоек** (Runner/Combat)

- Эффективность зависит от **правильного выбора инструмента** (оружие = ключ, контент = замок)

  

### Два столпа прогрессии:

1. **Персонажи (Pawns)** = Потенциал (перки, статы, навыки)

2. **Экипировка (Gear)** = Реализация (доступ к контенту, выживание)

  

**Правило:** Топовый персонаж без gear = беспомощен. Серый персонаж с топ gear = опасен.

## 3. Игровой Цикл (Core Loop)

  

### 1. Подготовка (Хаб)

* Выбор персонажа из **Колоды** (Roster)

* Подбор экипировки под задачу:

  - **Оружие:** Tier должен совпадать с уровнем монстров (Penetration Gating)

  - **Броня:** Баланс защиты и карманов (больше пластин = меньше лута)

  - **Противогаз:** Доступ к токсичным зонам (Environmental Gating)

* Проверка **"Шумности"** (Stealth Value): Слишком дорогая экипировка привлекает Охотников

  

### 2. Вылазка (Рейд)

**Стойка "Беглец" (Runner Stance):**

- Быстрое передвижение (100% скорость)

- Быстрый лут (1 сек на контейнер)

- **НО:** Беззащитен (оружие в ножнах)

  

**Стойка "Дуэлянт" (Combat Stance):**

- Медленное передвижение (60% скорость)

- Доступен бой (мили, стрельба, блок)

- Пластины автоматически защищают vital zones

  

**Переход между стойками:** 0.5 сек (уязвимое окно)

  

**Тактика:**

- Осторожные стычки с игроками (поиск уязвимостей в броне)

- Силовое подавление монстров (проверка tier оружия)

  

### 3. Эвакуация

* Выход через Убежища (если нет Метки)

* Или Ритуалы (для тяжёлого груза, например, Крио-капсул с новыми персонажами)

  

### 4. Прогресс

* Конвертация лута в **Резы** (валюта)

* Прокачка репутации **Фракций** (доступ к информации об аномалиях)

* Вскрытие **Крио-капсул** → Пополнение Колоды

## 4. Боевая Система: Stance Combat

  

### Философия

В Элдрейне бой — это **управление уязвимостью**, а не просто нажатие на курок.

  

### Две стойки

  

#### Стойка "Беглец" (Runner)

| Параметр | Значение |

|:---------|:---------|

| Скорость | 100% |

| Атака | Невозможна |

| Блок | Невозможен |

| Лут | Быстрый (1 сек) |

| Стамина | Восстанавливается |

  

**Использование:**

- Грабёж контейнеров на бегу (Munchkin mode)

- Бегство из невыгодного боя

- Паркур (прыжки через препятствия)

  

**Риск:** Если настигли в этой стойке, не успеешь перейти в боевую (анимация 0.5с = смерть)

  

---

  

#### Стойка "Дуэлянт" (Combat)

| Параметр | Значение |

|:---------|:---------|

| Скорость | 60% |

| Атака | 100% (мили + огнестрел) |

| Блок | Доступен (RMB) |

| Лут | Медленный (3 сек) |

| Стамина | Восстановление -50% |

  

**Механика Auto-Tanking (FPS решение):**

При зажатии `RMB` (блок):

1. Камера сдвигается влево/вправо (5-10°)

2. Модель персонажа разворачивает бронированное плечо к центру экрана

3. На HUD появляется силуэт пластины с индикатором прочности

4. Попадание в пластину = 0 HP урона + Impact (отброс + потеря прицела)

  

**Использование:**

- Ближний бой (commitment-based melee)

- Прицельная стрельба (ADS)

- Танкинг урона (подставить плечо)

  

**Риск:** Медленная мишень для снайперов

  

---

  

### Баллистика: PvP vs PvE

  

#### PvP (Игрок vs Игрок)

**Цель:** Gap Shooting (попадание в щели между пластинами)

  

| Урон по... | Результат |

|:-----------|:----------|

| **Пластине** | 0 HP урона, Impact (отброс), потеря Stamina |

| **Ткани (Гамбезон)** | -20% урона (ткань впитала пулю) |

| **Голому телу** | 100% урона + Кровотечение |

  

**Тактика:**

- Огнестрел: стреляй в голову/шею/подмышки

- Мили: сбей Impact'ом → добей, когда враг на земле

  

---

  

#### PvE (Игрок vs Монстр)

**Цель:** Tier-Gating (оружие = ключ к контенту)

  

Монстры имеют **Hide Density** (плотность шкуры):

  

| Монстр | Tier | Требуется оружие | Без правильного Tier |

|:-------|:-----|:-----------------|:---------------------|

| Крыса | T1 | Любое | Норм урон |

| Мясник | T3 | T3+ | Пули застревают (-90% урон) |

| Демон | T5 | T5 артефакт | Не пробивает вообще |

  

**Исключение:** Холодное оружие не имеет Tier-гейтинга против плоти, но требует Blunt Damage против костяной брони (скелеты, големы)

## 5. Система Брони: Композит

  

### Структура защиты

  

| Слой | Материал | Функция | Покрытие |

|:-----|:---------|:--------|:---------|

| **Базовый** | Мешковатая ткань (Гамбезон) | % резист + карманы | 100% тела |

| **Защитный** | Пластины (плечи, грудь, руки) | Полная неуязвимость зоны | 30-70% (по тиру) |

  

### Тиры Брони

  

| Tier | Покрытие пластинами | Ткань (резист) | Карманы | Визуал |

|:-----|:-------------------|:---------------|:--------|:-------|

| T1 | 30% (только грудь) | -10% урон | 6 слотов | Нагрудник + тряпки |

| T2 | 50% (грудь + плечи) | -20% урон | 8 слотов | Наплечники + пояс |

| T3 | 70% (+ руки) | -30% урон | 10 слотов | Полный доспех |

  

**Правило "Стойки и Покрытия":**

- **Беглец:** Пластины защищают **спину** (бежишь от врага)

- **Дуэлянт:** Пластины защищают **грудь** (лицом к врагу)

## 6. Система Персонажей: Колода (Deck)

  

### Философия

Игрок — это **Кукловод** (Puppeteer). Персонажи — это **Пешки** (Pawns).

  

### Lifecycle персонажа

1. **Находка:** Крио-капсулы спавнятся в рейдах (занимают 3x5 слотов рюкзака)

2. **Вскрытие (Хаб):** Игрок видит:

   - Расу (Лазур, Урсарид, Вермин, Канид)

   - Трейты (случайные мутации, например, "Ночное зрение")

   - Класс (Страж, Дуэлянт, Охотник, Алхимик)

3. **Решение:**

   - Принять → добавить в Колоду

   - Разобрать → получить ресурс для апгрейда других

   - Продать → выставить на Аукцион

  

### Симбиоз: Персонаж + Gear

  

**Правило "Голого Героя":**

Персонаж без экипировки = беспомощный

  

| Без... | Последствие |

|:-------|:------------|

| Оружия | Не может убить T2+ монстров |

| Брони | Умирает от 2-3 ударов |

| Батарей | Нет маны для скиллов |

| Противогаза | Не может зайти в токсичные зоны |

  

**Почему это не P2W:**

- Богач купил топ персонажа → Но без топ gear он слаб

- Бедняк с серым персонажем + топ gear → Может победить богача

  

### Смерть и Permadeath

* **KIA:** Снаряжение теряется, тело остаётся на карте

* **Missing:** Если тело не эвакуировали союзники, персонаж уда
