# Реестр: Ключевые Точки (POIs & Buildings)

> **Логика:** Здесь описаны уникальные локации, здания и данжи.
> **Types:** `Safe Zone`, `Shop`, `Event`, `Dungeon`.
> **Access:** `Open` (Открыто), `Key` (Нужен ключ), `Timed` (По времени), `Cycle` (Зависит от ротации секторов).

---

## Сектор 01: Ржавый Порт (Port POIs)

### "Дверь" (The Door)
[poi:: port_door]
*Древний причал с аркой из вибрирующей пустоты.*
- **Тип:** `Safe Zone` / `Spawn Point`
- **Доступ:** `Open` (Всегда доступно, сектор статичен).
- **Функция:** Точка входа для новых игроков ("Осколков").
- **NPC:** Маяк Приветствия (Инструктаж).

### Арена "Перекресток Судеб" (Crossroads Arena)
[poi:: port_arena]
*Круглая площадка на воде, окруженная трибунами из лодок.*
- **Тип:** `PvP/PvE Event`
- **Доступ:** `Open`
- **Функция:** Дуэли игроков, испытания билдов.
- **Награда:** Репутация фракции *Бойцовский Клуб*.

### Черный Рынок "Сердце" (The Heart Market)
[poi:: port_black_market]
*Скрытые торговые ряды в затопленных подвалах.*
- **Тип:** `Shop (Hidden)`
- **Доступ:** `Timed` (Только ночью).
- **Функция:** Продажа контрабанды, стимуляторов, информации о данжах.
- **Владелец:** Теневые Торговцы.

## Подземелья Порта (Port Dungeons)

### Старый Крематорий "Тень Смерти"
[poi:: dungeon_crematorium]
*Мрачное кирпичное здание с высокими трубами.*
- **Тип:** `Dungeon` (Horror / Loot-run)
- **Сложность:** `Medium`
- **Враги:** Призраки, Оживший Пепел.
- **Механика:** Дебафф "Удушье" (постоянный урон без противогаза).
- **Лут:** Урны с прахом, Некромантские артефакты.

### Пирс "Тропы Преображения"
[poi:: dungeon_pier_paths]
*Разрушенный мост, уходящий в туманную Пустоту.*
- **Тип:** `Dungeon` (Parkour / Puzzle)
- **Сложность:** `High` (Skill-based)
- **Механика:** Прыжки по парящим платформам. Падение = Смерть группы.
- **Лут:** Высокий шанс нахождения Батарей в конце пути.

### Мистический Театр "Парадокс"
[poi:: dungeon_theatre_paradox]
*Полузатопленное здание оперы.*
- **Тип:** `Dungeon` (Wave Defense / Time Anomaly)
- **Сложность:** `Variable`
- **Механика:** "Эхо" игроков (встреча с копиями других групп из прошлого). Режим "Спектакль" (5 волн врагов).
- **Лут:** Театральный реквизит (ценный хлам), маски.

### Храм Морского Бога
[poi:: dungeon_sea_temple]
*Логово босса, скрытое в подводной пещере.*
- **Тип:** `Boss Lair`
- **Владелец:** [mob:: mired_lord]|[faction:: deep_lash]
- **Лут:** Уникальное оружие, [item:: black_heart].
