# Система: Репутация (Standing)

## 1. Уровни Доверия (Tiers)
Репутация измеряется от -1000 до +1000.

| Tier | Название | Бонусы |
| :--- | :--- | :--- |
| **Enemy** | < -500 | NPC фракции атакуют в рейдах. Торговцы не говорят с вами. |
| **Neutral** | 0 | Базовые цены. Доступ к T1 товарам. |
| **Friendly** | 300+ | Скидка 10%. Доступ к T2-T3 товарам. |
| **Ally** | 800+ | Скидка 20%. Снижение налога на Аукционе. Доступ к T5 (Экзотика). |

## 2. Грайнд Репутации
Как поднять статус:
1.  **Сдача Трофеев:** Сдавать "Дог-таги" (жетоны) врагов фракции.
2.  **Контракты:** Ежедневные задания (Daily Quests).
3.  **Эксклюзив:** Продажа редких Артефактов конкретному торговцу (а не в общий Терминал).
