# Сервисы Города

## 1. Мед-Отсек (Sanatorium)
* **NPC:** Доктор Вэй.
* **Функция:**
    * Лечение травм (Переломы, Ожоги), которые не проходят сами.
    * Реабилитация "Поломок" (персонажей, вернувшихся из Missing In Action).
    * Установка Имплантов.

## 2. Мастерская (Workshop)
* **NPC:** Главный Механик.
* **Функция:**
    * **Ремонт:** Восстановление прочности за Резы.
    * **Калибровка:** Снижение шанса клина оружия (Jamming).
    * **Крафт:** Создание патронов и расходников из мусора.

## 3. Терминал Наймы (Recruitment)
* **Функция:** Покупка новых капсул с персонажами (Гача-механика).
* **Ротация:** Список доступных рекрутов обновляется раз в 24 часа.

## 4. Склад (Storage)
Личный сундук игрока.
* Общий для всех персонажей аккаунта.
* Расширяемый за валюту.
