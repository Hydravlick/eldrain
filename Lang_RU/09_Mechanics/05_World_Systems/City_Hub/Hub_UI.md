# Интерфейс: Хаб (Hub UI)

## 1. Философия
Интерфейс Хаба должен быть диегетическим, где это возможно.
* Вместо кнопки "Магазин" — игрок подходит к прилавку.
* Вместо меню "Рейд" — игрок подходит к воротам шлюза или карте.

## 2. Экран Подготовки (Loadout Screen)
Самый важный экран перед выходом.
* **Центр:** 3D модель персонажа.
* **Справа:** Инвентарь Склада (Stash).
* **Слева:** Инвентарь Персонажа (Equipped).
* **Чеклист:** Система автоматически предупреждает, если вы забыли важное:
    * *"Нет патронов для основного оружия!"*
    * *"Нет еды!"*
    * *"Сломана броня!"*

## 3. Экран Эвакуации (Post-Raid)
Показывается после успешного выхода.
1.  **Loot Tally:** Список вынесенного. Подсчет общей стоимости.
2.  **XP/Mastery:** (Скрытый прогресс владения оружием).
3.  **Status Update:** "Персонаж устал" / "Получил травму".
