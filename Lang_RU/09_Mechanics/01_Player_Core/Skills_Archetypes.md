# Система Навыков и Архетипов

## 1. Структура Слотов (2+1)
Каждый персонаж обладает жестким лимитом активных способностей для сохранения тактической ясности.

* **2 Активных Навыка (Active Slots):**
    * Используют ресурсы: Энергию ([[System_Batteries]]) или Расходники ([[Consumables_Rules]]).
    * Имеют Cooldown.
    * Привязаны к клавишам `Q` и `E`.
* **1 Пассивный Навык (Trait Slot):**
    * Постоянный эффект или аура.
    * Не требует ресурсов.
    * Определяет стиль игры (например, "Регенерация при беге" или "Урон из тени").

## 2. Формирование Навыков
Навыки не выбираются из общего дерева, а **складываются** из тегов сущности:
* `Раса` (Дает пассивный Trait).
* `Специализация` (Дает первый Активный Навык).
* `Экипировка/Артефакт` (Может дать второй Активный Навык или модификатор).

## 3. Архетипы Навыков
Функциональные категории для баланса.

### Атака (Offense)
* **DirectDamage (DD):** Точечный урон (Skillshot).
* **AreaDamage (AoE):** Урон по площади, высокий расход маны.
* **Trap (Ловушки):** Контроль территории, отложенная активация.

### Контроль и Поддержка (Support)
* **Control (CC):** Стан, замедление, ослепление.
* **Buff/Debuff:** Модификация статов союзников или врагов.
* **Heal:** Восстановление HP (Редкий ресурс, доступен только медикам и через дорогие расходники).

### Утилиты (Utility)
* **Mobility:** Рывки, телепорты, крюк-кошка.
* **Defense:** Персональные щиты, стены.

## 4. Ролевая Матрица
| Роль | Комбинация Архетипов | Задача |
| :--- | :--- | :--- |
| **Assault (Штурм)** | DD + Mobility | Агрессия, флангование. |
| **Tank (Танк)** | Defense + Control | Впитывание урона, инициация. |
| **Support (Саппорт)** | Heal + Buff | Поддержка живучести отряда. |
| **Scout (Разведка)** | Trap + Mobility | Разведка, контроль путей отхода. |
