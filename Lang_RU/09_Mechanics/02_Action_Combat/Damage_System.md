# Система: Расчет Урона (Damage)

## 1. Базовая Формула
Урон в Элдрейне не рандомный (нет разброса 1-10). Это детерминированная система "Ключ-Замок".

$$Damage = (Base\_Dmg * Velocity\_Mod) - (Armor\_Mitigation)$$

* **Base_Dmg:** Урон оружия.
* **Velocity_Mod:** Бонус скорости (для персонажей с высокой `AGI` или при разгоне).
* **Armor_Mitigation:** Снижение урона броней.

## 2. Механика Брони: Threshold vs Resistance

### A. Threshold (Порог Пробития) — The "Ping" Effect
Самый важный параметр. Если оружие не может пробить броню, оно наносит 0 урона (Рикошет).
* **Условие:** Если `Weapon.Penetration` < `Target.Armor_Tier`:
    * **Результат:** Урон = 0 (или 1 дробящего, "Stun").
    * **VFX:** Искры, звук металла, враг не замедляется.

### B. Resistance (Сопротивление)
Если пробитие успешно (`Penetration` >= `Armor_Tier`), вступает в силу снижение урона.
* Каждый класс брони имеет `% поглощения`.
    * *Light Armor:* -10% входящего урона.
    * *Heavy Plate:* -50% входящего урона (но требует высокого Tier пробития).

## 3. Типы Урона
Тип урона определяет, какой статус накладывается при попадании (см. [[Status_Effects]]).

| Тип | Эффект | Эффективность против |
| :--- | :--- | :--- |
| **Physical (Kinetic)** | Обычный урон. | Слабая броня. |
| **Thermal (Fire)** | Накладывает `[Burn]`. Разрушает прочность брони. | Живая плоть, Дерево. |
| **Entropy (Void)** | Игнорирует броню, но имеет низкий базовый урон. | Тяжелая броня, Щиты. |
| **Chemical (Acid)** | Накладывает `[Corrosion]`. Снижает Armor Tier цели. | Механизмы. |

## 4. Зоны Поражения (Hitboxes)
* **Голова:** x2.0 Урон. (Если нет шлема соответствующего Тира).
* **Тело:** x1.0 Урон.
* **Конечности:** x0.7 Урон + Шанс на `[Cripple]` (Хромота/Перелом).
