# Механика: Пороги Угрозы (Threat Thresholds)

## 1. Расчет Угрозы
Суммируется `CurrentResonance` всех предметов в инвентаре (с учетом множителей x1.0 и x0.25). Полученное число сравнивается с **Лимитом Аномалии** (зависит от сложности локации).

Это не проверка силы снаряжения для выживания в новом Tier. За жесткий фильтр среды отвечает [[08_World_Generation/Generation/08_Gate_Check|Gate Check]].

- **Gate Check:** слишком слабый комплект погибает при Phase Shift.
- **Threat Threshold:** слишком дорогой/чужеродный комплект привлекает Аномалию или не допускается на вход.

Обе проверки выводятся вместе как `Readiness Corridor`: нижняя граница - выживаемость среды, верхняя - допустимый резонанс. Игрок должен видеть, что он может быть одновременно "слишком слабым для T2" и "слишком громким для T1".

## 2. Уровни Агрессии (Traffic Light System)

| Уровень            | % от Лимита | Реакция Мира                                                                                      |
| :----------------- | :---------: | :------------------------------------------------------------------------------------------------ |
| **Тишина (Green)** |   0 - 90%   | Базовый спавн. Монстры агрятся только по зрению/слуху.                                            |
| **Звон (Yellow)**  |  90 - 110%  | Расширенный радиус агро. Монстры чувствуют игрока сквозь стены в радиусе 20м.                     |
| **Охота (Red)**    | 110 - 150%  | **Спавн Охотников.** В радиусе 100м периодически появляются элитные мобы, знающие позицию игрока. |
| **Cингулярность**  |    >150%    | **Вход невозможен**.<br>**Метка:** Большинство точек эвакуации для Вас отключены. Режим хардкор+  |
 
## 3. Отказ в Доступе (Entry Denial)
* **Условие:** `Total_Resonance > 150%` от Лимита Локации.
* **Эффект:** Игрок **не может войти** в рейд. Аномалия отторгает столь сильный диссонанс.
* **Значение:** Это балансировщик, запрещающий игрокам в топовой экипировке фармить низкоуровневые зоны.
* *Исключение:* Внутри рейда превышение 150% возможно (через сбор лута), это переводит игру в режим "Сингулярность" (экстремальное выживание), но не выкидывает игрока.
