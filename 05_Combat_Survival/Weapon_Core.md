# Оружие: Магострельный Канон и Тиры

## 1. Главный принцип

Дальний бой в Элдрейне - это **магострельная технология**: катушки, руны, конденсаторы, тросы, батареи, перегрев и резонанс.

```text
Дальний бой = давление + контроль + окно добивания
не = непрерывный DPS и низкий TTK
```

Оружие может убивать самостоятельно, если игрок держит дистанцию, контролирует охлаждение, тратит заряды и попадает в слабые места. Но нормальная роль магострела - сбить темп, раскрыть позицию, прожечь защиту или создать окно для `melee / Q / E / команды`.

## 2. Запрет на современный темп

В каноне MVP нет скоростных магазинных фреймов и непрерывного дальнего DPS. Быстрый повторный огонь заменяется на:

- `charge_time` - подготовка импульса;
- `bloom` - эфирный разброс после движения, паники и повторного выстрела;
- `heat` - перегрев ствола, катушки или перчатки;
- `resonance` - шум для Аномалии;
- `battery_cycle` - 1-3 заряда до замены батареи.

Если способность временно повышает темп, она должна платить перегревом, резонансом, потерей движения или расходом всех оставшихся зарядов.

## 3. Иерархия Тиров

Tier задает не эпоху оружия, а качество стабилизации импульса.

| Tier | Класс | Примеры | Игровой смысл |
|:---|:---|:---|:---|
| **T1** | Scrap Frame | самодельный разрядник, игольник, кривой эмиттер | высокий bloom, грубый импульс, дешевый вход |
| **T2** | Guild Frame | гильдейский конденсатор, портовый гарпун, стабильный фокус | нормальный рейдовый стандарт |
| **T3** | Expedition Frame | усиленная катушка, герметичный эмиттер, боевой фокус | лучше держит heat, меньше дрейфует |
| **T4** | Alter Frame | рунический контур, живой металл, Reality Burn канал | сильные эффекты, высокий резонанс |
| **T5** | Relic Frame | уникальный довоенный артефакт | меняет правила фрейма, требует риска |

## 4. Основные фреймы дальнего боя

| Frame | Роль | Боевой ритм |
|:---|:---|:---|
| `handcannon` | тяжелый одиночный разряд, stagger, добивание | прицел -> импульс -> откат |
| `condenser_longframe` | дальний заряженный выстрел, weakspot, щиты | заряд -> выстрел -> охлаждение |
| `scatter_emitter` | ближний веер энергии, контроль входа | накачка -> выброс -> перегрев |
| `harpoon_driver` | фиксация цели, tether, вытягивание из укрытия | заряд -> попадание -> трос |
| `needle_crossbow` | тихий механический выстрел без батареи | взвод -> болт -> долгая перезарядка |
| `catalyst_focus` | чистый эфир, Reality Burn, ритуалы | `Reality Charge` -> канал -> импульс -> backlash |

## 5. Тиры, TTK и броня

Высокий TTK держится не тем, что все выстрелы слабые, а тем, что выстрел редко является бесплатным продолжением мышки.

- Хорошее попадание должно быть ощутимым: stagger, aim punch, трещина щита, открытие weakspot.
- Убийство дальним боем требует серии решений: позиция, заряд, расход батареи, контроль heat, повторный угол.
- Ближний бой и способности остаются важными, потому что магострел чаще **создает окно**, чем закрывает весь бой сам.
- `Efficient Tier` по-прежнему нужен для PvE-пробития, но он не должен превращать оружие в непрерывный дальний DPS.

## 6. T.O.U.C.H. в оружии

| Атрибут | Влияние |
|:---|:---|
| `TRQ` | удержание тяжелого фрейма, recoil damp, осадная стойка |
| `GRP` | замена батареи, взвод, точность рук, контроль дрейфа |
| `LYR` | переносимость отдачи, ожогов, backlash и aim punch |
| `GLW` | охлаждение, перегруз, экономия заряда, Reality Burn |
| `SNS` | чтение weakspot, предупреждение перегрева, прогноз дрейфа |

## 7. Стихии и импульсы

Для MVP стихия - это не отдельная RPS-игра поверх оружия и не список школ магии для билдов. Это свойство импульса, батарейного канала, биома или статусного эффекта:

- `thermal` - ожог, panic, высушивание спор;
- `shock` - interrupt, перегруз техники, цепные реакции во влажной среде;
- `acid` - raid debuff брони и укрытий;
- `aether` - щиты, Reality Burn, аномальные тела;
- `kinetic` - stagger, tether, aim punch, пролом.

Биом может усиливать или глушить импульсы, но базовый бой строится на фрейме, батарее, позиции, T.O.U.C.H. и окне добивания. Если стихии останутся только в лоре, эти поля все равно работают как описания статусов и окружения, не как обязательная отдельная прогрессия.
