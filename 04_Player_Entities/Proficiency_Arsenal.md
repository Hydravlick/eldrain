# Адаптивный Арсенал и Мастерство

## 1. Адаптивный Арсенал (Weapon Gating)
У персонажа нет навыка "Владение всем". Есть **Белый Список (Whitelist)** — `Allowed_Weapons`, который формируется при генерации.

$$Allowed = (RaceList \cup SpecList) - RaceBanned$$

1.  **Race List:** То, что существо может держать физически (Ёж может держать бревно, Крыса — нет).
2.  **Spec List:** То, чему существо обучено (Солдат умеет стрелять из винтовки).
3.  **Race Banned:** То, что существо НЕ может использовать, даже если Класс позволяет (например, `[Small Frame]` запрещает `[Heavy]` оружие).

**MVP-источник правды:** пока RaceList/SpecList не вынесены в отдельные поля рас и классов, итоговый `Allowed Arsenal` читается из `Registry_Combos`. Это сохраняет текущую систему, где P/Q/E и арсенал уже описаны вместе как результат смешения расы и специализации.

## 2. Система Мастерства (Proficiency Tiers)
Вместо жесткого запрета "Нельзя", используется система **Soft Gating** (Мягкое ограничение) через уровни владения.

**Формула:** `Базовое владение (Раса/Класс) + Теги (Чипы/Мутации) = Итоговый уровень`.

| Уровень (Tier) | Статус | Эффект в игре |
| :--- | :--- | :--- |
| **0** | **Чужак** | **Блокировка.** Персонаж не может экипировать предмет. |
| **1** | **Новичок** | **Штраф.** Можно экипировать, но использование неэффективно (штрафы к анимации/отдаче). |
| **2** | **Профи** | **Норма.** Базовое состояние для "родного" класса. Оружие работает на 100%. |
| **3+** | **Мастер** | **Гейт открыт.** Оружие становится продолжением тела и включает скрытый `weapon_vector`. |


## 3. Расширение Арсенала
Игрок может обойти ограничения и повысить уровень владения, устанавливая **Теги** (Импланты/Мутации).
* *Пример:* Тег `[Servo-Arm]` позволяет Крысе игнорировать запрет на `[Heavy]` оружие.
* *Пример:* Тег `[Veteran Soldier]` дает `+1 к Магострелу`, превращая Новичка в Профи.

## 4. Вектор оружия как гейт, а не постоянная стихия

Оружие не добавляет вектор в Combat Profile само по себе. Оно делает это только если персонаж достиг нужного уровня владения.

```text
final_prof = combo_prof + tag_prof_delta + temporary_modifiers
if final_prof >= vector_gate:
    active_vectors += weapon_vector
```

Рекомендуемый MVP:

| Тип оружия | `weapon_vector` | `vector_gate` | Смысл |
|:---|:---|:---:|:---|
| `blade` | `shadow` | 3 | На мастерстве клинок становится инструментом засад, обхода и точного входа. |
| `blunt` | `kinetics` | 3 | На мастерстве дробящее оружие ломает механизмы, стойки и темп. |
| `polearm` | `kinetics` | 3 | На мастерстве древковое оружие контролирует дистанцию массой и рычагом. |
| `arcanegun` | `ballistics` | 3 | На мастерстве магострел дает огневое подавление и темп. |
| `catalyst` | `aether` | 3 | На мастерстве катализатор открывает эфирные перегрузки. |
| `shield` | `kinetics` | 3 | На мастерстве щит становится активным инструментом тарана и позиционного давления. |

До Tier 3 оружие может быть сильным численно, но не участвует в Двойном Парадоксе как отдельный вектор. Это защищает идентичность `Race x Spec`.
