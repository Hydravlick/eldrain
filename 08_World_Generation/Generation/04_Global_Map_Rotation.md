# Система: Ротация Районов (District Rotation)

## 1. Концепция "Колоды"
География Элдрейна нестатична и состоит из **7 Ключевых Секторов** (Районов), например: Порт, Собор, Фабрики, Депо. Однако игроки не имеют доступа ко всем районам одновременно.

## 2. Состояния Сектора
Каждый сектор может находиться в одном из двух состояний:
* **Активен (Active/Cursed):** В секторе бушует Аномалия. Он наполнен Эфиром, лутом и монстрами. Вход открыт для рейдов.
* **Покой (Resting/Stable):** Аномалия ушла. Уровень Эфира слишком низок для формирования ценного лута. Вход для рейдеров закрыт (или не имеет геймплейного смысла).

## 3. Серверная Синхронизация
Состояние мира едино для всех игроков (или шарда).
* **Независимость:** Ротация происходит на сервере и не зависит от действий конкретного игрока.
* **Адаптация:** Игрок подстраивается под расписание мира, а не мир под игрока.

## 4. Сдвинутый Пул Тиров
Активная колода поддерживает три параллельные локации:

- одна свежая или ранняя T1;
- одна зрелая T2;
- одна умирающая T3;
- опционально четвертая ивентовая/overflow-локация.

Когда T3-локация коллапсирует, она возвращается в колоду как новый T1. Это сохраняет FOMO и миграцию игроков, но не наказывает часовой пояс: в любой момент есть понятная точка входа для слабого, среднего и сильного комплекта.

Прямая точка входа не означает бесплатный Tier. T2/T3 открываются через Access Contracts: маршрут, цена, подготовка среды, лимит Диссонанса и свободный слот сервера. Это сохраняет позднюю фазу населенной без скрытого матчмейкинга по стоимости билда.
