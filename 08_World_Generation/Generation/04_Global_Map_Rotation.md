# Система: Ротация Районов (District Rotation)

## 1. Концепция "Колоды"
География Элдрейна нестатична и состоит из **7 Ключевых Секторов** (Районов), например: Порт, Собор, Фабрики, Депо. Однако игроки не имеют доступа ко всем районам одновременно.

## 2. Состояния Сектора
Каждый сектор может находиться в одном из двух состояний:
* **Активен (Active/Cursed):** В секторе бушует Аномалия. Он наполнен Эфиром, лутом и монстрами. Вход открыт для рейдов.
* **Покой (Resting/Stable):** Аномалия ушла, сектор физически стабилизирован, уровень Эфира слишком низок для формирования ценного лута. Вход для свободного перемещения закрыт; игрок видит сектор как мирную проекцию на столе и работает с доступными POI через интерфейс.

## 3. Серверная Синхронизация
Состояние мира едино для всех игроков (или шарда).
* **Независимость:** Ротация происходит на сервере и не зависит от действий конкретного игрока.
* **Адаптация:** Игрок подстраивается под расписание мира, а не мир под игрока.

## 4. Сдвинутый Пул Тиров
Активная колода поддерживает три параллельные локации:

- одна проявляющаяся или ранняя T1;
- одна вспоминающая катастрофу T2;
- одна пересобираемая T3;
- опционально четвертая ивентовая/overflow-локация.

Когда T3-локация проходит финальную Стабилизацию, её инстанс закрывается, а временная Stable-конфигурация возвращается в городскую колоду. Снимок завершённой генерации становится мирной проекцией на [[08_World_Generation/Hub/01_Hub_Map_Table|Живой Миниатюре]]. Позднее новая Аномалия замещает сектор, открывает новый T1 и после следующей Стабилизации заменяет геометрию проекции. Это сохраняет FOMO и миграцию игроков, но не наказывает часовой пояс: в любой момент есть понятная точка входа для слабого, среднего и сильного комплекта.

Стабилизация сектора не является безопасной выдачей оставленного лута. Личная добыча закрепляется только успешной экстракцией по [[06_Economy_Loot/Extraction_Stabilization_Loop|общему контракту]].

Прямая точка входа не означает бесплатный Tier. T2/T3 открываются через Access Contracts: маршрут, цена, подготовка среды, лимит Диссонанса и свободный слот сервера. Это сохраняет позднюю фазу населенной без скрытого матчмейкинга по стоимости билда.
