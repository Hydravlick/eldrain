# Реестр: Глобальный бестиарий

> **Логика:** Здесь описаны мобы всех локаций, их варианты, поведение и таблицы лута. Конкретный сектор получает пул через `location_tags`, `ecology_layer` и условные теги линии.
> **Stats:** `HP` (Здоровье), `AR` (Броня/Резист), `SPD` (Скорость м/с).
> **AI Tags:** `[behavior:: aggressive]` (преследование), `[behavior:: defensive]` (удержание дистанции), `[behavior:: swarm]` (стая). Технические legacy-ID могут сохранять старое слово, но не являются игроко-видимым названием существа.

## Слои экосистемы

- `[ecology_layer:: baseline]` — базовый пул указанных локаций.
- `[ecology_layer:: mutation_line]` — отдельный моб активной линии.
- `[ecology_layer:: mutation_variant]` — разрешённый вариант базового моба.
- `[ecology_layer:: encounter]` — босс, событие или отдельная встреча.
- `[ecology_layer:: unassigned]` — определение без активного пула.

`[location_tags:: port|factory]` перечисляет совместимые локации через `|`. `[mutation_line:: none]` используется для независимого от линии контента; моб линии хранит точный ID линии и стадии.

Народное имя не обязано объяснять истинную природу существа. Жители называют моба по тому, что видно с улицы: мешкам на голове, звону в воде, лишней пасти или чистому следу после кормёжки.

## Dataview: сводка мобов

```dataviewjs
const registryPath = dv.current().file.path;
const content = await dv.io.load(registryPath);
const blocks = content.split(/^### /m).slice(1);

function field(block, key) {
    const match = block.match(new RegExp(`\\[${key}::\\s*([^\\]]+)\\]`, "i"));
    return match ? match[1].trim() : "";
}

const rows = blocks.map(block => {
    const header = block.split("\n")[0].trim();
    const mobId = field(block, "mob");
    const variantId = field(block, "variant_id");
    if ((!mobId && !variantId) || mobId === "template_mob") return null;
    return [
        `[[${registryPath}#${header}|${header.replace(/\s*\(.*$/, "")}]]`,
        variantId ? "вариант" : "моб",
        field(block, "location_tags").replaceAll("|", ", ") || "—",
        field(block, "ecology_layer") || "—",
        field(block, "mutation_line") || "—",
        field(block, "mutation_stage") || field(block, "mutation_stages") || "—",
        field(block, "threat_tier") || "—",
        field(block, "balance_state") || "—"
    ];
}).filter(Boolean)
  .sort((a, b) => a[2].localeCompare(b[2]) || a[3].localeCompare(b[3]) || a[0].localeCompare(b[0]));

dv.table(["Определение", "Тип", "Локации", "Слой", "Линия", "Стадия", "Tier", "Баланс"], rows);
```
