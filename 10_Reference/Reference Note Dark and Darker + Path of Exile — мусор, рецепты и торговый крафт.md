# Dark and Darker + Path of Exile: мусор, рецепты и торговый крафт

> Авторское предпочтение: для Eldraine приятнее именно DaD-подход, где ценность рождается из вылазки, торговца, найденного мусора и конкретного рецепта. PoE полезен не как тон или темп, а как экономическая логика: почти любой материал может быть не "мусором на продажу", а будущим рычагом аффикса, реролла, базы или рецепта.

## 1. Почему это отдельный референс

В уже существующих заметках есть крафт, аффиксы и бартер. Эта запись уточняет вкус системы:

```text
не магазин с валютой
не чистый random loot
не бесконечный склад бесполезного мусора

а:

локационный мусор
+ торговец/мастер
+ рецепт
+ катализатор
+ аффиксный результат
= осмысленный extraction-крафт
```

Цель: игрок должен радоваться странной находке не потому, что она дорого продаётся, а потому что он знает или подозревает, кому она нужна и какой тип предмета может из неё родиться.

## 2. Что приятно в Dark and Darker

DaD силён тем, что торговцы и квесты превращают dungeon-мелочи в цели следующего рейда:

- предмет должен быть добыт в рейде или иметь нужный статус;
- торговец просит конкретные части монстров, расходники, трофеи или действия;
- affinity открывает товары, скидки и крафт;
- квесты ведут игрока в определённые карты, модули и типы столкновений;
- даже простая вещь получает память: "это нужно Алхимику", "это пригодится Оружейнику", "это часть цепочки".

Для Eldraine это лучше, чем абстрактная фракционная репутация. Игроку приятнее, когда он выносит не "10 единиц ресурса", а предмет с адресом:

```text
сломанный фильтр из T2-канализации
-> нужен Портовому Механику
-> вместе с влажной медью и Резом даёт маску с environment-аффиксом
```

## 3. Что взять из Path of Exile

PoE полезен своей экономической идеей: currency items имеют собственное применение, а не являются только деньгами. В Eldraine это переводится так:

- `Flux` не просто ресурс, а способ добавить случайную настройку;
- `Essence` не просто редкий лут, а гарантия семейства аффикса;
- `Entropy Orb` не просто рискованный материал, а ставка на Corruption;
- blueprint не просто unlock, а знание о допустимой трансформации;
- мусор из конкретной локации может быть частью рецепта, потому что несёт средовой след.

Не нужно копировать орбы PoE буквально. Нужно взять принцип:

```text
каждый crafting material является действием,
а не только ценой.
```

## 4. Предпочтительный цикл Eldraine

```text
1. Игрок находит странный junk / trophy / trace / broken device.
2. Предмет имеет источник: локация, фаза, моб, материал, температура, влажность, фракционный след.
3. В Хабе торговец или ночной workbench распознаёт его как ингредиент.
4. RecipeTransaction принимает:
   Inputs + Blueprint + Catalyst_Rez + Decorators
5. Результат получает базу, допустимые аффиксные семейства и риск Defect/Corruption.
6. Игрок получает новую причину идти в конкретную локацию за недостающим материалом.
```

Важно: рецепт не обязан быть полностью известен сразу. Можно показывать "силуэт" результата:

- известен торговец;
- известна категория предмета;
- известно одно семейство аффикса;
- неизвестен точный диапазон;
- неизвестен риск дефекта, пока не найден лучший blueprint или мастер.

## 5. Ночные workbench-рецепты

Отдельная приятная идея: в ночных, опасных, полуразрушенных мастерских внутри рейда могут находиться конкретные рецепты.

Такие рецепты не должны быть обычным UI-меню. Они должны быть частью вылазки:

- верстак стоит в T2/T3, в тёмной мастерской, лаборатории, кузне или аномальном узле;
- рецепт можно выучить, скопировать, вынести как blueprint или использовать на месте;
- некоторые рецепты требуют локального условия: холод, влажность, активная лампа, энтропийный шторм, кровь моба, работающий генератор;
- использование workbench занимает время, шумит и создаёт риск;
- быстрый крафт на месте может дать предмет с Defect, а вынесенный blueprint позволяет сделать стабильную версию в Хабе.

Пример:

```text
Night Workbench: "Сырая медная прививка"
Локация: затопленный T2 архив
Inputs: wet_copper_scrap + cracked_filter + frog_salt + Catalyst_Rez
Decorator: cooling_essence optional
Output: mask insert или light armor mod
Possible affix family: environment / reliability
Risk: damp_defect, если верстак не стабилизирован
```

## 6. Торговцы как преобразователи, не магазины

Торговец в Eldraine должен быть не списком товаров, а преобразователем мира:

| Тип торговца | Что принимает | Что создаёт | Игровое поведение |
|---|---|---|---|
| Механик | скрап, фильтры, пружины, корпуса | маски, инструменты, reliability-аффиксы | игрок ищет техно-мусор |
| Алхимик | соли, жидкости, органы, грибы | расходники, токсин-контроль, cantrip-снижение | игрок лезет во влажные/ядовитые зоны |
| Оружейник | Frame-обломки, пластины, батарейные гнёзда | оружие, cycle/handling-аффиксы | игрок охотится за боевыми остатками |
| Портной/бронник | ткань, кожа, ремни, теплоизоляция | броневые Pattern и mobility/environment | игрок выносит "низкий" бытовой лут |
| Архивариус | Trace, записи, реликвии, слепки | blueprint, lore unlock, special recipe | игрок сохраняет странные небоевые находки |
| Ночной мастер | опасные материалы и локальный workbench | редкий рецепт, Defect/Corruption-крафт | игрок рискует крафтить внутри рейда |

Так мусор разных локаций становится адресным:

```text
не "продать всё серое"
а "оставить серое, потому что это кормит мой следующий билд"
```

## 7. Аффиксы через рецепты

Связка с [[07_Gear_Inventory/Affix_Grammar|Affix_Grammar]]:

- обычный рецепт создаёт базовый предмет без аффикса или с безопасным малым свойством;
- `Flux` добавляет случайное допустимое семейство;
- `Essence` гарантирует семейство: `cycle`, `handling`, `signature`, `reliability`, `environment`, `body_interface`;
- локационный ингредиент может сузить пул;
- мастерская может добавить Defect из-за условий;
- ночной workbench может открыть Corruption-ветку, но с риском потери базы.

Пример:

```text
wet_copper_scrap из затопленной зоны
  -> повышает шанс environment / reliability

burnt_anchor_glass из T3 шторма
  -> открывает signature / body_interface
  -> повышает DissonanceLoad результата

ratwork_spring из техтоннеля
  -> handling / cycle
  -> может дать defect: jitter
```

## 8. Ограничения против абьюза

| Риск | Метод абьюза | Ограничение |
|---|---|---|
| игрок хранит весь мусор навсегда | схрон превращается в свалку | категории, теги, фракционные заказы, сжатие материалов после стабилизации |
| один рецепт становится лучшим фармом | все бегают в одну локацию | ротация заказов, лимит workbench, переменные условия карты |
| дешёвый мусор даёт топ-аффиксы | бюджетный крафт ломает прогрессию | сильные аффиксы требуют опасный источник, blueprint, мастерство или T2/T3 условие |
| ночной workbench фармят без риска | игроки зачищают и крафтят безопасно | шум, время, нестабильность, деградация света, ограниченный заряд верстака |
| PoE-сложность душит extraction | игрок читает таблицы вместо мира | рецепты группировать по торговцам и понятным семействам, не по сотням скрытых тегов |

## 9. Что важно для ощущения игрока

Игрок должен думать:

- "это пока мусор, но я знаю, кому он нужен";
- "этот предмет пахнет конкретной локацией";
- "если найду рецепт, эта дрянь станет частью билда";
- "можно рискнуть и крафтить ночью прямо в рейде";
- "аффикс родился из истории предмета, а не выпал из пустой таблицы".

Это поддерживает scavenger-fantasy лучше, чем чистая продажа лута за Рез.

## 10. Короткий вывод для Eldraine

Базовая формула:

```text
DaD-приятность:
  конкретный торговец
  + конкретный вынесенный мусор
  + affinity / услуга / рецепт

PoE-полезность:
  материал является действием
  + аффикс управляется катализатором
  + редкость не равна простому урону

Eldraine-версия:
  локационный мусор
  + фракционный мастер
  + RecipeTransaction
  + Decorators
  + Night Workbench risk
  = предмет с памятью вылазки
```
