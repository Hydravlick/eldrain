# Игровой Цикл

Каноническая версия игрового цикла находится в [[01_Core_Vision/02_Core_Loop]].

Короткая формула:

```text
Хаб -> Выбор цели и коридора допуска -> Рейд -> Точка жадности -> Gate Check или эвакуация -> Стабилизация -> Новый рейд
```

Старое правило "рейд = 30-45 минут" заменено на гибкий личный темп внутри 6-часовой жизни локации:

- 10-15 минут - короткая работа.
- 20-30 минут - стандартный рейд.
- 35-45 минут - глубокая вылазка.
- 45+ минут - фазовый серфинг через Gate Check.
